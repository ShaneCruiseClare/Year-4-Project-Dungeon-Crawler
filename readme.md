This is a working prototype of A-star path finding.
The project get a text file that generates and draw a 3d map layout 
from the text file for each level. 

- Each level is given a tile type for the entry, exit, and key of the level.
- The A-Star finds a path from the entry to exit and gives back the cheapest
  and fastest route. The result is printed in the console.
